# Eclipse 꿀팁 정리

Java 수업과 실습을 하면서 자주 사용하는 Eclipse 단축키와 기능을 정리한 문서입니다.

---

## 자주 쓰는 단축키

| 기능                 | 단축키                     |
| ------------------ | ----------------------- |
| 자동완성               | `Ctrl + Space`          |
| 코드 자동 정렬           | `Ctrl + Shift + F`      |
| import 자동 정리       | `Ctrl + Shift + O`      |
| 실행                 | `Ctrl + F11`            |
| 주석 처리 / 해제         | `Ctrl + /`              |
| 빠른 수정              | `Ctrl + 1`              |
| 이름 한 번에 변경         | `Alt + Shift + R`       |
| Getter / Setter 생성 | `Alt + Shift + S` → `R` |
| 생성자 생성             | `Alt + Shift + S` → `O` |
| 프로젝트 새로고침          | `F5`                    |
| 파일 저장              | `Ctrl + S`              |
| 전체 파일 저장           | `Ctrl + Shift + S`      |
| 찾기                 | `Ctrl + F`              |
| 전체 검색              | `Ctrl + H`              |
| 한 줄 삭제             | `Ctrl + D`              |
| 한 줄 복사             | `Ctrl + Alt + ↓`        |
| 한 줄 이동             | `Alt + ↑ / ↓`           |

---

## 자동완성 관련 팁

* `main` 입력 후 `Ctrl + Space`를 누르면 main 메서드를 자동 생성할 수 있다.
* `sysout` 입력 후 `Ctrl + Space`를 누르면 출력문을 자동 생성할 수 있다.
* 클래스명, 메서드명, 변수명을 끝까지 입력하지 않아도 자동완성으로 빠르게 작성할 수 있다.

---

## 코드 정리 팁

* 코드가 지저분해졌을 때는 `Ctrl + Shift + F`로 자동 정렬한다.
* import 오류가 생기면 `Ctrl + Shift + O`로 import를 정리한다.
* 오류가 난 부분에서는 `Ctrl + 1`을 눌러 Eclipse가 추천하는 해결 방법을 확인한다.

---

## 클래스 작업 팁

* 필드를 만든 뒤 Getter / Setter는 직접 치지 말고 자동 생성 기능을 사용한다.
* 생성자도 직접 치기보다 자동 생성 기능을 사용하면 오타를 줄일 수 있다.
* 변수명이나 클래스명을 바꿀 때는 직접 하나씩 고치지 말고 `Alt + Shift + R`로 한 번에 변경한다.

---

## 실행 관련 팁

* 실행은 `Ctrl + F11`로 빠르게 할 수 있다.
* 콘솔창이 안 보이면 `Window → Show View → Console`에서 다시 열 수 있다.
* 코드가 맞는데 오류가 계속 남아 있으면 `Project → Clean`을 실행해본다.
* 외부에서 파일을 추가했는데 Eclipse에 안 보이면 `F5`로 새로고침한다.

---

## 실습할 때 기억할 점

* 파일명과 `public class` 이름은 같아야 한다.
* `package` 이름이 다르면 같은 클래스처럼 보여도 접근이 안 될 수 있다.
* 저장하지 않은 파일 때문에 오류가 남아 있을 수 있으니 자주 저장한다.
* 같은 이름의 클래스가 여러 개 있으면 다른 클래스를 참조할 수 있으니 위치를 확인한다.
* 오류 메시지를 먼저 읽고, 빨간 줄에서 `Ctrl + 1`을 눌러 해결 힌트를 확인한다.

---

## 자주 쓰는 핵심 단축키 3개

| 기능        | 단축키                |
| --------- | ------------------ |
| 자동완성      | `Ctrl + Space`     |
| 코드 정렬     | `Ctrl + Shift + F` |
| import 정리 | `Ctrl + Shift + O` |

이 세 가지는 Eclipse에서 Java 코드를 작성할 때 가장 자주 사용하는 기본 단축키이다.
