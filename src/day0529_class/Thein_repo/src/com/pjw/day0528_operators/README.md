# 0528 연산자 operators

## 연산자 종류

| 종류 | 의미 |
|---|---|
| 산술연산자 | `+`, `-`, `*`, `/`, `%` |
| 증감연산자 | `++`, `--` |
| 할당연산자 | `=`, `+=`, `-=`, `*=`, `/=` |
| 비교연산자 | `>`, `<`, `>=`, `<=`, `==`, `!=` |
| 논리연산자 | `&&`, `||`, `!` |
| 비트연산자 | 비트 단위 계산 |

## 제일 헷갈리는 포인트

- `++i`는 먼저 증가하고 출력
- `i++`는 먼저 출력하고 나중에 증가
